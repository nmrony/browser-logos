## How do I get this logo?

You can either:

* Install it using

  * [`npm`][npm]: `npm install --save-dev @browser-logos/android-webview-beta`
  * [`yarn`][yarn]: `yarn add --dev @browser-logos/android-webview-beta`

* Use [`cdnjs`][cdnjs].
