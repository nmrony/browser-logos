## How do I get this logo?

You can either:

* Install it using

  * [`npm`][npm]: `npm install --save-dev @browser-logos/opera-developer`
  * [`yarn`][yarn]: `yarn add --dev @browser-logos/opera-developer`

* Use [`cdnjs`][cdnjs].
