## How do I get this logo?

You can either:

* Install it using

  * [`npm`][npm]: `npm install --save-dev @browser-logos/firefox-aurora_5-22`
  * [`yarn`][yarn]: `yarn add --dev @browser-logos/firefox-aurora_5-22`

* Use [`cdnjs`](https://cdnjs.com/libraries/browser-logos)
