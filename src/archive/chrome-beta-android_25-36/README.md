## How do I get this logo?

You can either:

* Install it using

  * [`npm`][npm]: `npm install --save-dev @browser-logos/chrome-beta-android_25-36`
  * [`yarn`][yarn]: `yarn add --dev @browser-logos/chrome-beta-android_25-36`

* Use [`cdnjs`][cdnjs].
